HOW TO USE THESE FILES:

1. Create a new folder: journey-connects/

2. Download/copy all files to this structure

3. Run:

4. Test locally:
   npm run dev
   
5. Build for production:
   npm run build
   
6. Push to GitHub and deploy to Cloudflare Pages (see README.md)
